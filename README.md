# How to use?
## Prereqisitions
```
npm install --save-dev @deep-foundation/npm-automation
```
## Using
```
npx npm-pull
```
```
npx npm-release [--new-version=<NEW_VERSION>] [--package-json-file-path=<PACKAGE_JSON_FILE_PATH>] [--deep-json-file-path=<DEEP_JSON_FILE_PATH>]
```
